# Getting Started with smart Calculator

Calculator is one of the most banal projects.

## Why a calculator?

For the first application on React, I decided to choose the creation of a calculator. This is a great idea for leveling up skills. 

## Stack:
React / JS / Chakra UI / Axios/ Cheerio /

### general: 
 👉 use `Chakra UI`  to design the interface
 👉 use functional components in `React`

**Note: This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).**

## What did I do?

### calculator
✔️ data entry from the clickboard
✔️ data entry from the keyboard
✔️ calculations history
✔️ simple implementation of the ability to use the latest calculations
✔️ calculates in automatic mode when entering expressions into input
✔️ working with local storage and API
✔️ used regular expressions
✔️ used Reack Hooks - `useEffect` for automatic focus on the input field

### convector units of measurement
Working with arrays and data objects.

### currency convector
I used `Axios`  to pull the layout from the site and used `Cheerio`  to filter quotes from the data received.
